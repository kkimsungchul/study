# 적용방법
	※ 안됨, 버전이 Ruby 버전이 안맞아서 번들 인스톨 자체가 안됨..


	1. https://github.com/theorydb/theorydb.github.io 접속
		※ 우측상단에  star 누르자
	
	2. download.zip 으로 코드파일을 내려받기

	3. 압축풀기

	4. 해당 디렉토리에서 "bundle install" 명령어 입력

	5. bundle exec jekyll serve 명령어 실행
