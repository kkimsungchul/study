[ XSS 필터 사용방법 ]
	


# 1. 조치할 프로젝트 내의 JAVA 디렉토리에 하단의 파일을 추가
	* 패키지를 생성하고 넣어준다. 
	RequestWrapperXSS.java
	XssFilter.java	
	
	ex)
		sampleSite 라는 프로젝트에 해당 필터 추가
		경로 : SampleSite03\src\main\java\com\sungchul\filter
		

# 2. web.xml 파일에 필터 추가
	* class 의 경로는 프로젝트 상황에 맞게 맞춰주면 됨
	* URL 맵핑은 모든URL 에 대해서 맵핑함

	================================================================================
    <filter>
        <filter-name>XssFilter</filter-name>
        <filter-class>filter.XssFilter</filter-class>
    </filter>
    <filter-mapping>
        <filter-name>XssFilter</filter-name>
        <url-pattern>/*</url-pattern>
    </filter-mapping>
	================================================================================
