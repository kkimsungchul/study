1. 자체 솔루션(통합계정관리) 관리툴 개발 - 시큐러스
설명 : 여러 개의 시스템에 있는 사용자들의 정보를 하나의 시스템에서 관리하는 솔루션입니다.
개발 환경 : Spring / JSP  사용
업무 참여 내용 :
* 데이터베이스 설계 진행
* 화면단  개발 진행 (JQuery , bootstrap  사용)
* 서버단  개발 진행 (MVC 패턴 , Ajax를  통한 비동기 처리)
* 기존에 존재하던 관리툴을  제거하고 본인이 만든 관리툴이  솔루션에 포함되었으며, 확장성 및 유지보수가 능하도록 개발하여 다양한 고객사의 시스템에 이식이 가능하도록  수정

2. 자체 솔루션 구축 진행 - 시큐러스
기존에 회사에 존재하던 솔루션 및 신규 개발한 솔루션을 고객사에 구축 진행하였습니다.

2-1. SSO솔루션( Single Sign-ON, 통합인증) 고객사 구축 진행 - 시큐러스
설명 : 고객사의 여러시스템들의 로그인 로직을 분석 한뒤에 SSO솔루션으로 시스템을을 연동하는 작업입니다.
개발 환경 : ASP , PHP , JSP(Spring 포함)
업무 참여 내용 : 
* ASP , PHP , JSP 등의 다양한 시스템들을 분석 
* 분석한 시스템에 솔루션을 적용할 수 있도록 해당시스템 및 솔루션 수정작업 진행
* 기존 시스템에서 필요없는 로직 및 데이터베이스 필드 삭제로 솔루션의 경량화 영향을 줌


2-2. 통합계정관리 솔루션 고객사 구축 진행 - 시큐러스
설명 : 여러개의 시스템에 나눠져 있는 사용자의 정보를 취합하여 한곳에서 관리하는 솔루션입니다.
개발 환경 : Spring , Java , Tibero , MSSQL , Windows AD(엑티브디렉토리)
연동 대상시스템 분석 및 통합계정관리 솔루션 구축 진행 
업무 참여 내용 : 
* 데이터베이스 설계 참여 
* AD연동 모듈 수정
* Tibero 연동 모듈 수정
* 관리툴 개발


3. 보안 SI (시큐어코딩, 서버취약점 조치)
설명 : 취약점 점검으로 검출된 웹 및 서버취약점들에 대한 보안 조치 SI입니다.
업무 참여 내역 :
* 약 30개 이상의 업체의 서버 및 웹취약점 조치 진행
* 리눅스 및 윈도우 서버 취약점 조치
* JAVA , ASP , PHP 의 웹 취약점 조치 진행
