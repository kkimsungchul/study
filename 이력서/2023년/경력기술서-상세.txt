# 보안 SI 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2018.03 ~ 2020.07
- 개요 : 보안 취약점 조치 SI
- 상세 내용 : 
해당 프로젝트는 웹 취약점 점검으로 검출된 보안 취약점들을 조치하는 프로젝트입니다.
입사 후 비정기적으로 보안취약점 조치 요청이 들어올 때마다 진행하였으며 짧게는 2주에서 길게는 한 달 이상의 기간 동안 작업을 진행했습니다.
저는 고객과의 커뮤니케이션, 취약점 조치를 담당했습니다.
JAVA/JSP, PHP, ASP로 개발된 약 30여개의 사이트를 행정안전부 보안취약점 28종에 검출되지 않도록 조치하였습니다.


# SSO 솔루션 구축 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2019.12 ~ 2020.02
- 개요 : SSO(Single Sign On) 솔루션을 고객사에 구축
- 상세 내용 : 
해당 프로젝트는 회사에서 보유하고 있는 SSO 솔루션을 구축하는 프로젝트입니다.
SSO는 Single Sign On으로 1회 인증으로 여러 시스템을 추가 인증 없이 이용할 수 있는 통합 인증 기능입니다.
고객과의 커뮤니케이션, 고객사 시스템 분석, 솔루션 커스터마이징, 솔루션 구축을 담당하였습니다.
솔루션은 JAVA로 개발된 기본적인 모듈만 있기에 각각의 시스템에서 작동하기 위해서는 해당 시스템의 환경을 토대로 솔루션 커스터마이징을 진행했습니다.
구축을 위해 ASP 1개, PHP 2개로 이뤄진 고객사의 시스템을 분석하였으며 기존에 사용하던 SNS 간편 로그인 기능을 사용하여서 로그인해도 SSO 기능을 사용할 수 있도록 구축하였습니다.

# 통합계정관리 솔루션 구축 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2020.05 ~ 2020.08
- 개요 : 통합계정관리 솔루션을 고객사에 구축
- 상세 내용 : 
해당 프로젝트는 회사에서 보유하고 있는 통합계정관리 솔루션을 구축하는 프로젝트입니다.
통합계정관리 솔루션은 시스템별로 분산된 사용자 계정정보를 통합적으로 관리하는 시스템입니다.
고객사에서 사용하던 기존 솔루션의 노후화 및 유지보수 불가로 인해 대체 할 수 있는 솔루션이 필요했기에 프로젝트를 진행하게 되었습니다.
저는 고객과의 커뮤니케이션, 기존에 사용하던 솔루션 분석, 연동 대상시스템(AD , 인사시스템) 분석, 솔루션 관리 툴 개발, 솔루션 구축을 담당하였습니다.
회사에는 기존에 솔루션 관리 툴이 없었기에 프로젝트 기간 동안 혼자 Spring/JSP를 사용하여 구축하였습니다.
솔루션 관리 툴에서는 인사시스템에서 연동 이뤄지지 않는 기타 사용자들에 대한 CRUD 기능과 연동 로그, AD의 특정 OU 에 사용자 추가 기능을 개발하였습니다
해당 솔루션 구축 후 고객사는 기존에 수작업으로 OU에 사용자 추가하는 것을 자동화하여 업무의 효율성을 높였고,
회사 내에서는 솔루션 관리 툴이 없었기에 고객사에서 시연자료나 솔루션 화면에 대한 자료 요청 시 보낼 정보가 없었지만 솔루션 관리 툴을 개발하여 홍보 및 시연자료를 만들 수 있었습니다.

# 해양경찰청 인사평가시스템 개발 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2020.10 ~ 2020.12
- 개요 : 해양경찰청의 인사평가 시스템 개발 참여
- 상세 내용 : 
인사발령부서에서 엑셀 파일로 수작업하여 관리하는 인사관리와 관련된 모든 업무를 시스템으로 구축하여 관리하는 시스템입니다.
해당 프로젝트는 Spring/JSP로 개발하였으며, 인사 발령 및 부서 내 조직도 관리, 데이터 통계 및 히스토리 기능을 개발을 담당하였습니다.
조직도는 계층 구조여서 생성 이동 삭제 시 고려해야 할 내용이 많기에 프로세스를 꼼꼼히 살피고 개발하여 정확한 조직도 관리가 되도록 하였습니다.
또한 신규 인사발령 데이터와 이전 인사발령 데이터를 토대로 경찰서 및 각 부서의 인원, 직급 등의 다양한 통계를 확인할 수 있는 기능을 개발하여 인사발령에 필요한 데이터들을 비교하기 쉽게 하였습니다


# 학점은행제 및 정보공시 시스템 고도화 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2020.12 ~ 2021.02
- 개요 : 학점은행제 및 정보공시 시스템 고도화 개발 참여
- 상세 내용 : 
해당 프로젝트는 기존에 Spring으로 개발하여 사용하던 시스템의 불편한 점을 개선 및 고도화하는 프로젝트입니다.
정보공시 시스템은 이미 구축되어 있었지만 사용자가 사용할 때 불편한 기능이 존재했습니다.
저는 기능 개선 개발을 주로 담당하였으며 고객의 여러 요구사항에 맞도록 시스템을 개선하였습니다.
교육기관 코드를 검색해서 입력하지 않고 사용자가 직접 입력하므로 인한 오류 발생, 초성 검색 기능의 부재, 정렬 및 수정 삭제 기능, 보안 취약점 조치 및 예외 처리 기능 개발을 하였습니다.
사용자의 실수로 인한 데이터 정정은 개발자가 직접 수정해야 하는 업무여서 고도화 진행 중에도 틈틈이 데이터 정정을 요청이 들어왔지만 고도화가 마무리되어 갈 무렵에는 데이터 정정을 요청하는 사용자들의 민원이 줄어든 것을 확인하였습니다.


# SSO솔루션구축 프로젝트 (추가 구축)
- 소속 : (주)시큐러스
- 기간 : 2021.03 ~ 2021.06
- 개요 : SSO(Single Sign On) 솔루션을 고객사에 구축
- 상세 내용 : 
기존 2019년 12월에 솔루션을 구축한 고객사에서 추가로 개발한 사이트에 대해 SSO 솔루션을 추가 구축한 프로젝트입니다.
이전과 같이 고객과의 커뮤니케이션, 고객사 시스템 분석, 솔루션 커스터마이징, 솔루션 구축을 담당하였습니다.
PHP로 개발된 신규 2개의 사이트의 시스템을 분석하고 SSO 솔루션을 구축하였고 기존에 사용하던 SNS 로그인 기능의 라이브러리 버전 업데이트하였습니다.



# 통합계정관리 솔루션 고도화 프로젝트
- 소속 : (주)시큐러스
- 기간 : 2021.06 ~ 2021.10
- 개요 : 기존에 개발한 통합계정관리 솔루션의 관리툴 고도화 프로젝트
- 상세 내용 : 
이전에 구축한 통합계정관리 관리 툴을 SpringBoot/thymeleaf로 고도화하여 구축하는 프로젝트입니다.
고객사에서 기존에 사용하던 계정관리 시스템의 기능이 정상 작동 하지 않아 통합계정관리 솔루션의 연동 기능을 제외한 하나의 DB만을 관리할 수 있는 솔루션 구축을 요청하였습니다.
저는 고객과의 커뮤니케이션, 대상 DB 분석, 솔루션 고도화, 솔루션 구축을 담당하였습니다.
다만 회사 내에 개발해놓았던 통합계정관리 관리 툴은 코스콤의 시스템의 맞게 개발해놓은 상태여서 고객사의 요구사항에 맞게 하려면 너무 많은 부분을 수정해야 했기에 고도화를 진행했습니다.
공통으로 필요한 기능과 추가로 필요한 기능을 식별하여 기존의 CSS와 재사용할 수 있는 부분의 소스를 가져와서 공통기능을 가지고 있는 관리 툴을 개발했습니다.
공통관리 툴 개발 완료 후 별도의 브런치에서 고객사의 시스템에 맞도록 관리 툴을 커스터마이징해 구축하였습니다.
새로운 관리 툴 개발로 추후 통합계정관리 솔루션 관리 툴 커스터마이징에 드는 공수를 줄였습니다.


# D-space 개발/운영 프로젝트
- 소속 : (주)지뉴소프트
- 기간 : 2021.10 ~ 현재
- 개요 : 다양한 개발 툴 (Jira , Gitlab , Jenkins , Sparrow) 을 하나의 시스템에서 관리하는 서비스 구축 / 운영
- 상세 내용 : 
해당 프로젝트는 다양한 개발 툴들을 사용자들이 하나의 포털에서 관리할 수 있도록 해주는 시스템 구축 프로젝트입니다.
프로젝트 투입 시 저의 주 업무는 Jenkins에서의 CRUD와 Build Log 조회 원격 빌드 기능 개발이었습니다.
포털 개발 완료 후 고도화가 바로 진행되었으며 고도화 시작 후 기존인력들의 퇴사로 인해 Jenkins 외에 Jira , Gitlab 의 연동 부분의 기능개선도 일부분 맡아 개발하였습니다.
또한 API로 여러 툴을 제어하다 보니 포털에서 CRUD를 실행하였지만 오류가 발생하여 각 툴에 정상적으로 반영되지 않은 경우에 대한 롤백 정책과 재동기화 정책을 정하고 그에 맞도록 개발을 진행했습니다.
추후 기존 PL의 타 부서 발령으로 인하여 2022년 5월부터 해당 프로젝트의 PL을 맡게 되어 추가 시스템(Sparrow)연동에 대한 DB 설계 및 연동 부분을 맡아 개발하였습니다.
PL을 맡은 이후 오픈소스인 Keycloak의 암호화 모듈 개발도 진행하였습니다.

[ 경력 에쎄이 ] 

2018년 3월에 보안솔루션 및 보안취약점 회사에 입사하여서 솔루션 구축 업무가 없는 동안은 여러 고객사 사이트의 보안취약점 조치를 진행했습니다.
ASP, PHP, JSP 로 개발된 웹 사이트들에서 XSS, SQL Injection, 데이터 평문전송 등 다양한 보안 취약점을 조치하며 타 사이트들의 개발된 코드들을 보면서 코드를 보는 눈을 키웠습니다.
다양한 사이트들의 취약점을 조치하다보니 다양한 취약점을 통해 서비스 장애 또는 정보 탈취등이 이뤄질수 있는지 이해했습니다.

2019년 12월부터 2020년 2월까지는 SSO솔루션 구축 프로젝트를 진행했습니다.
SSO(Single Sign On)는 1회 인증으로 여러 시스템을 추가 인증 없이 이용할 수 있는 통합 인증 기능입니다.
PHP로 개발된 3개의 사이트를 SSO솔루션으로 엮는 프로젝트에서 저는 시스템 환경정보 조사 및 솔루션 모듈 수정 , 각 사이트별 로그인 로직 수정을 담당했습니다.
보안취약점을 조치 할때는 소스반영방법이나 개발언어에 대해서만 알면 크게 문제가 되지 않았지만, 솔루션을 구축할때는 운영체제, DB , WEB , 방화벽 정책, 개발언어의 상세 버전 등 많은 정보가 필요했습니다.
어떤 환경 정보가 필요한지 몰랐던 저는 환경정보 조사할 때 여러번 환경정보에 대한 문의메일을 보내면서 어려워 했습니다. 다만 이러한 경험으로 환경정보 조사의 중요성을 알았습니다.

2020년 5월부터 2020년 8월까지는 통합계정관리 솔루션 구축을 했습니다.
통합계정관리 솔루션은 시스템별로 분산된 사용자 계정정보를 통합적으로 관리하는 시스템입니다.
인사시스템(Tibero)-> 통합계정관리시스템 -> AD 순서로 데이터가 전달되야 됐으며 솔루션은 중간 단계에 구축 하게 됐습니다.
이 당시만 해도 제대로된 솔루션의 매니저화면도 없던 터라 솔루션의 매니저화면 개발, 시스템 구축의 업무를 동시에 수행했습니다.
